GTK

GTK est une bibliothèque open source disponible sur crates.io. Il s’agit d’une bibliothèque qui permet de créer des interfaces graphiques pour une application. Les interfaces peuvent recevoir des entrées d’utilisateur. Notre application a été confectionnée avec plusieurs bibliothèques qui fonctionnent en cohésion avec GTK.

Lien de la bibliothèque

Gdk-Pixbuf

Gdk-pixbuf est une dépendance de GTK qui nous permet de créer, modifier ou charger des images. Cette bibliothèque est également open source et nous l’avons utilisé avec Cairo-rs pour dessiner des formes plus facilement.

Lien de la bibliothèque

Cairo

Cairo-rs est une bibliothèque open source qui permet de dessiner des formes, écrire sur des surfaces et les enregistrer avec Gdk-Pixbuf ou les afficher avec Gtk-Image.

Lien de la bibliothèque

L’ensemble du projet a été réalisé sous Rust 1.69 et est compatible avec les versions ultérieures.
